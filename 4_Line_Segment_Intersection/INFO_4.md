# 선분 교차 판정

## 정의
2차원 평면 위에서 두 선분이 주어졌을 때, 이 두 선분이 서로 교차하는지 아닌지를 판정하는 문제이다.

여기서 교차는 다음 경우를 모두 포함한다.

- 두 선분이 내부에서 서로 만나는 경우
- 한 끝점에서만 만나는 경우
- 두 선분이 일직선 위에 있고 일부 구간이 겹치는 경우

## 핵심 개념: CCW
세 점 `p1`, `p2`, `p3`가 있을 때, 이 세 점의 방향성을 `CCW(Counter Clock Wise)`로 판단할 수 있다.

좌표를 각각 `(x1, y1)`, `(x2, y2)`, `(x3, y3)`라고 하면 다음 값을 계산한다.

`S = (x1y2 + x2y3 + x3y1) - (y1x2 + y2x3 + y3x1)`

이 값의 부호에 따라:

- `S > 0`: 반시계 방향
- `S < 0`: 시계 방향
- `S = 0`: 일직선

## 선분 교차 판정 원리
선분 `p1-p2`와 선분 `p3-p4`가 있을 때,

- `ccw(p1, p2, p3) * ccw(p1, p2, p4)`
- `ccw(p3, p4, p1) * ccw(p3, p4, p2)`

를 계산한다.

일반적인 경우에는 이 두 값이 모두 `0 이하`이면 두 선분은 교차한다.

## 예외: 일직선 위에 있는 경우
모든 CCW 값이 `0`이면 두 선분은 같은 직선 위에 있다.

이 경우에는 방향성만으로는 교차 여부를 알 수 없고, 실제로 구간이 겹치는지를 따로 확인해야 한다.

즉:

- 두 선분의 끝점을 정렬한 뒤
- 구간이 서로 겹치는지 확인해야 한다.

## 알고리즘
1. `ccw(p1, p2, p3)`와 `ccw(p1, p2, p4)`를 계산한다.
2. `ccw(p3, p4, p1)`와 `ccw(p3, p4, p2)`를 계산한다.
3. 두 곱이 모두 `0 이하`인지 확인한다.
4. 단, 모두 `0`인 경우에는 구간 겹침 여부를 따로 확인한다.

## 시간 복잡도
모든 계산은 상수 번의 정수 연산만 필요하므로 `O(1)`이다.
