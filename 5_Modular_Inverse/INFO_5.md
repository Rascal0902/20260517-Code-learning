# 소수 모듈러에서의 잉여역수

## 정의
정수 `a`와 양의 정수 `p`에 대하여,

`a * x ≡ 1 (mod p)`

를 만족하는 정수 `x`를 `a`의 `mod p`에서의 잉여역수 또는 모듈러 역원이라고 한다.

이 문제에서는 `p`가 소수이고 `1 <= a < p`라고 가정한다.

## 왜 항상 존재하는가
`p`가 소수이고 `1 <= a < p`이면 `gcd(a, p) = 1`이다.  
즉 `a`와 `p`는 서로소이므로 `mod p`에서 역원이 항상 존재한다.

## 페르마의 소정리
`p`가 소수이고 `a`가 `p`의 배수가 아니면

`a^(p-1) ≡ 1 (mod p)`

가 성립한다.

양변에 `a^(-1)`을 곱하는 관점으로 보면

`a^(p-2) ≡ a^(-1) (mod p)`

이므로, 잉여역수는 `a^(p-2) mod p`를 계산하면 된다.

## 왜 단순 곱셈은 느린가
`a^(p-2)`를 그대로 `a`를 `p-2`번 곱해서 구하면 `O(p)` 시간이 걸린다.  
`p`가 매우 크면 이 방법은 비효율적이다.

## 분할 정복 거듭제곱
거듭제곱은 다음 성질을 이용해 빠르게 계산할 수 있다.

- `n`이 짝수이면 `a^n = (a^(n/2))^2`
- `n`이 홀수이면 `a^n = a * a^(n-1)`

이를 이용하면 지수를 절반씩 줄일 수 있어서 시간 복잡도는 `O(log p)`가 된다.

## 알고리즘
1. 입력으로 `a`, `p`를 받는다.
2. `p`가 소수이고 `1 <= a < p`라고 가정한다.
3. `a^(p-2) mod p`를 빠른 거듭제곱으로 계산한다.
4. 그 값을 출력한다.

## 시간 복잡도
빠른 거듭제곱을 사용하면 `O(log p)`이다.
